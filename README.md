**Midterm Examination Reflection**
What is Git used for?
Git is used for tracking changes in files, especially in coding projects. It helps developers save different versions of their work so they can go back if something goes wrong.
What is the difference between Git and GitHub?
Git is a tool installed on your computer that manages version control locally. GitHub is an online platform where you can store and share your Git projects with others.
Why is branching useful?
Branching is useful because it allows you to work on new features or fixes without affecting the main project. This makes development safer and more organized.
What command did you use to upload your local commits to GitHub?
The command used to upload local commits to GitHub is git push origin main. This sends my changes from my computer to the remote repository on GitHub.
What did you learn from this activity?
I learned how to properly use Git commands for saving and managing project versions. I also learned how to connect my local repository to GitHub and push my work online.
